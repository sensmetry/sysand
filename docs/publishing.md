# Publishing a Package to Sysand Package Index

Since Sysand v0.1.0, self-serve package publishing is supported through the
[publish][pub_comm] command and the official [Sysand Package Index][sysand_index].

To learn how to publish a package on the official index, follow its
documentation: [Tutorial: Publishing Your First Project][publish_tutorial].

[pub_comm]: ./commands/publish.md
[sysand_index]: https://sysand.com
[publish_tutorial]: https://docs.sysand.com/tutorial/publishing-your-first-project
