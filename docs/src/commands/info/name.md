# `sysand info name`

Get or set the name of the project

## Usage

```sh
sysand info name [OPTIONS]
```

## Description

Prints the name of the given project. With `--set`, updates the name.

The `name` field is required and cannot be cleared.

## Options

- `--set <NAME>`: Set the project name
