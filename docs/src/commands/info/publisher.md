# `sysand info publisher`

Get or set the publisher of the project

## Usage

```sh
sysand info publisher [OPTIONS]
```

## Description

Prints the publisher of the given project. With `--set` or `--clear`, modifies the publisher.

## Options

- `--set <PUBLISHER>`: Set the project publisher
- `--clear`: Remove the project publisher
