## Dependency options

- `--index [<INDEX>...]`: URLs for indexes to use when resolving dependencies,
  in addition to the default indexes.
- `--default-index [<DEFAULT_INDEX>...]`: Set and override URL:s of the default
  indexes (by default '[https://beta.sysand.org](https://beta.sysand.org)')
- `--no-index`: Do not use any index when resolving this usage
- `--include-std`: Include usages of KerML/SysML v2 standard libraries if present
