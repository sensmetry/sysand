## Global options

- `-v`, `--verbose`: Use verbose output
- `-q`, `--quiet`: Do not output log messages
- `--no-config`: Disable discovery of configuration files
- `--config-file <CONFIG_FILE>`: Give path to 'sysand.toml' to use for configuration
- `-h`, `--help`: Print help
